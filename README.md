# User_Storys

How It Works

MAIN

The program asks the user to enter the product name.

It calls a function to check that the product price is a valid number greater than 0.

It calls another function to check that the product quantity is a valid integer greater than 0.

The program calculates the total price using this formula:

total_price = price * quantity

Finally, it prints all the product information.

Functions

verify_product_price()

This function asks the user to enter the product price.

It checks if the value is a number.

It checks if the number is greater than 0.

If the value is not valid, the program asks the user to try again.

verify_product_quantity()

This function asks the user to enter the product quantity.

It checks if the value is an integer.

It checks if the number is greater than 0.

If the value is not valid, the program asks the user to try again.
